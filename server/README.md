# graphql-demo: Server

### dependencies

```
npm i -S apollo-server-express body-parser cors express graphql graphql-tools node-fetch uuid
```

### Compiling

```
npm run start
```

For the REST wraper
```
npm run start:rest
```

Go to http://localhost:3000/graphiql to see GraphiQL
